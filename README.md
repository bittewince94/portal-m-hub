# portal-m-hub

## 项目简介

portal-m-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定域名或网站，仅作为 HTML 静态资源的集中存放与分发点。

## 目录说明

```
portal-m-hub/
├── pages/            # 存放各独立 HTML 页面
├── assets/           # 公共资源文件（CSS、JS、图片等）
└── README.md         # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，命名清晰，便于检索。
- `assets/`：存放页面间通用的静态资源，避免重复引用。

## 页面归档说明

本仓库的所有页面均为独立 HTML 文件，不依赖后端服务。每个页面可独立访问，适合作为轻量级展示页、工具页或信息页使用。

归档遵循以下原则：
- 页面内容不包含任何具体域名、网址或营销信息。
- 页面之间无强制跳转或依赖关系。
- 所有页面均采用标准 HTML5 规范编写。

## 维护说明

- 新增页面：在 `pages/` 目录下创建新文件夹或文件，确保内部资源路径正确。
- 更新页面：直接修改对应 HTML 文件，并同步更新 `assets/` 中的公共资源。
- 删除页面：移除对应目录或文件，并检查是否有残留引用。

欢迎提交 Pull Request 或 Issue 协助维护，但请确保内容符合项目定位。

## 许可

本项目所有内容仅供学习与归档使用。
